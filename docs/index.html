Zathura Liberation Day: Utility Plugin Support

A major breakthrough for Zathura customization, the "Zathura Liberation Day" patch introduces a powerful utility plugin system. This enables developers to create plugins that register new utility functions, extending Zathura's core capabilities. As part of this, a new TTS (Text-to-Speech) API is now available, allowing for the creation of plugins that can read documents aloud.

This functionality is currently available as a patch. You can find more details in the Zathura Pull Request #780.
